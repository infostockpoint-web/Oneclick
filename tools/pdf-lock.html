PDF Lock/Unlock

Add or remove password protection from PDFs. This typically requires a PDF encryption engine; here’s a polished workflow and guidance.

About locking/unlocking
  • Setting/removing PDF passwords in-browser requires encryption support (AES-128/256) at the PDF object stream level.
  • We plan to integrate a WebAssembly PDF toolkit for fully client‑side, private processing.
Until then
  1. Use a trusted desktop tool (Adobe Acrobat, qpdf, etc.)
  2. qpdf examples:
    qpdf --encrypt yourpass yourpass 256 -- input.pdf output-locked.pdf
    qpdf --password=yourpass --decrypt input-locked.pdf output.pdf
